Setup Development Environment

In order to kickstart the development of your web app with BRIDLE, the very first thing you need to do is to setup a react development environment.

Following tools are needed to setup a react dev environment:

  • Node JS

    • NodeJS works as a node package manager behind the scene. It is recommended to download and install the latest version of Node JS from its official site http://nodejs.org/

  • Yarn

    • Download the latest version of the most loved dependency management tool for react applications from its official site https://yarnpkg.com/ and install it on your machine.

Preparation

You'll need to install Node.js >=v8.11.4 (Recommended Version) (NPM comes along with it) and serve (a NPM package)

Use terminal and navigate to the project root.

Then run : npm install

Then run : npm start

It'll return something like this:

Serving!

- Local: http://localhost:3000

- On Your Network: http://192.168.1.18:3000

Build

Run to build the project : npm run build The build artifacts will be stored in the build/ directory.

Static Server

For environments using Node, the easiest way to handle this would be to install serve and let it handle the rest:

npm install -g serve

serve -s build

The last command shown above will serve your static site on the port 5000. Like many of serve’s internal settings

It'll return something like this:

Serving!

- Local: http://localhost:5000

- On Your Network: http://192.168.1.18:5000

Navigation

In the Top Navbar:

This functions used for state sideMenu true or false 


state = {
    sideMenu: false,
    ...
};

_toggleClass = () => {
    const currentSideMenu = this.state.sideMenu;
    this.setState({sideMenu: !currentSideMenu});
    this.props.onClick(this.state.sideMenu);
}

Because based on this.state.sideMenu

We can identify the classess for toggle, such as below: 


<div className={`burger-menu ${this.state.sideMenu ? 'toggle-menu' : ''}`} onClick={this._toggleClass}>

Change Layout

The template sidebar menu default is dark layout, if you want to change this with light default then you have to follow this steps

Step 1:

In the Navigation: which is placed in src/components/Navigation/Navigation.js

Comment out this import statement

import SideMenuDark from './SideMenu/SideMenuDark';

Step 2:

Remove the menuColor state from state

Step 3:

Remove the arrow class which is onSideMenuHandler

Step 4:

Replace the component with ternary JSX syntax from below of Navigation class

Step 5 & finally:

Remove the swither component from below of Navigation class
